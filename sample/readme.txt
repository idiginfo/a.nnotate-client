A.nnotate API sample code
=========================

This directory includes PHP sample code for accessing the
A.nnotate API.  

The 'index.html' in this directory includes instructions for using 
the PHP samples.

See http://a.nnotate.com/api-reference.html for a description 
of the API.



For Java programmers...
-----------------------
The java/ subdirectory includes an example showing how to 
access the API from Java; edit the settings in 
  java/AnnotateApi.java 
and 'make compile', 'make run' to run a test.



Please contact support@nnotate.com with any questions / problems 
/ suggestions using the API.
